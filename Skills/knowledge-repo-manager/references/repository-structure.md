# Repository Structure

- `Knowledge/INDEX.md`: main navigation.
- `Knowledge/REGISTRY.md`: project registry.
- `Knowledge/GOVERNANCE.md`: definitions and lifecycle rules.
- `Knowledge/projects/<project_id>/PROJECT_KNOWLEDGE.md`: canonical page per project.
- `Knowledge/templates/`: templates and snippets.
- `scripts/knowledge_validate.py`: structural validator.
- `scripts/knowledge_export.py`: markdown -> JSON exporter.
