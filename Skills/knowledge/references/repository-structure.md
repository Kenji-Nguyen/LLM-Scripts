# Repository Structure

- `docs/INDEX.md` — top-level navigation
- `docs/REGISTRY.md` — project registry
- `docs/GOVERNANCE.md` — governance rules and frontmatter spec
- `docs/ARCHITECTURE_PLAN.md` — architecture and rollout plan
- `docs/WORKFLOWS.md` — workflow playbook
- `knowledge/projects/<project_id>/PROJECT_KNOWLEDGE.md` — canonical project page
- `knowledge/templates/PROJECT_KNOWLEDGE.template.md` — project page template
- `knowledge/templates/SECTION_SNIPPETS.md` — section format snippets
- `knowledge/templates/INTERVIEW_INGEST.template.md` — ingest template
- `scripts/knowledge_validate.py` — structural validator
- `scripts/knowledge_export.py` — markdown to JSON exporter
- `skills/` — skill definitions (shared by Claude Code and Codex)
